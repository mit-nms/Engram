# FrontierCS Benchmarks

100+ problem instances across systems, ML, security, and scientific computing. From the [Frontier-CS project](https://github.com/FrontierCS/Frontier-CS).

The problem set lives in a git submodule at [`frontier_cs_repo/`](frontier_cs_repo/). Problems are split into two tracks:

- **[`algorithmic/`](algorithmic/)** -- Algorithmic problems (`--problem_name fcs_alg_<id>`)
- **[`research/`](research/)** -- Research problems (`--problem_name fcs_res_<id>`)
